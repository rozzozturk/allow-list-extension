#### Full Description

**Keepnet Assistant – Office 365 Allow List Configuration Assistant**

Keepnet Assistant is a powerful Chrome extension that automates allow list configuration for Keepnet phishing simulations in Office 365 environments.

**Features**

- **Automatic Navigation**: Automatic navigation in Microsoft Security Center and Exchange Admin Portal.  
- **Smart Clicking**: Element detection and automatic clicking system.  
- **Real-time Validation**: Automatic checks and validation at each step.  
- **Screenshot Evidence**: Automatic screenshot capture for each step.  
- **Multiple Workflow Support**: Automatic configuration for 6 different workflow types.  

**Supported Workflows**

1. **Third-Party Phishing Simulations** – Add phishing simulation domains and IPs to the allow list.  
2. **Anti-Spam Policies** – Connection Filter Policy configuration.  
3. **Safe Links** – Safe Links policy creation and domain addition.  
4. **Spam Filter Bypass** – IP-based bypass via Exchange transport rules.  
5. **ATP Link Bypass** – Advanced Threat Protection link bypass rules.  
6. **ATP Attachment Bypass** – Advanced Threat Protection attachment bypass rules.  

**Security & Privacy**

- All data is stored **locally only** on the user’s device.  
- No data is sent to external servers.  
- The extension works **only** on Microsoft Office 365 pages.  
- Source code is open and can be reviewed.  

**Usage**

1. Navigate to Microsoft Security Center (`https://security.microsoft.com`) or Exchange Admin Portal (`https://admin.exchange.microsoft.com`).  
2. Click the extension icon.  
3. The assistant panel opens on the page.  
4. Click the **Start** button to begin the workflow.  
5. Follow the step-by-step guidance shown in the panel.  

**Requirements**

- Office 365 admin account.  
- Access to Microsoft Security Center and Exchange Admin Portal.  
- Chrome browser.
